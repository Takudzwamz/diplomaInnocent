## Шаг 1 — Скачай обновление

Открой терминал в папке проекта и выполни:

```bash

git pull origin main
```

```bash

```

## Шаг 2 — Удали базу данных и перезапусти контейнеры

```bash
docker compose down
docker volume rm studentecommproject_sql-data
docker compose up -d
```

Подожди ~10 секунд, пока SQL Server запустится.

## Шаг 3 — Запусти приложение

```bash
cd StorefrontRazor
dotnet run
```

При первом запуске система автоматически создаст базу данных и заполнит её тестовыми данными. Это займёт ~20 секунд.

Адрес: **http://localhost:5249**

1. Открой **http://localhost:5249**

## Если диаграммы не нравятся

Все диаграммы — это код Mermaid прямо в `.md` файлах. Можешь:

1. Скопировать блок кода из любого документа
2. Вставить на сайт **https://mermaid.live**
3. Изменить и скачать новую картинку в PNG/SVG

Или запустить автоматическую генерацию всех диаграмм:

```bash
cd diplomnaya-rabota
python generate_diagrams.py
```

## Если что-то не работает

| Проблема | Решение |
|----------|---------|
| `dotnet run` — ошибка подключения к БД | Подожди 15 сек после `docker compose up -d` |
| Страница не открывается | Убедись что порт **5249** (не 5001!) |
| Нет рекомендаций на главной | Войди как пользователь (не аноним) |
| AI-рекомендации не появляются | Введи API-ключ в Admin → Settings |
| `git pull` конфликт | `git stash && git pull origin main && git stash pop` |
